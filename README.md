# weeker_raytracer

Based on v2 of https://github.com/RayTracing/raytracing.github.io, not the v3 under development

## Build

```
cd TheNextWeek
make clean && make debug
```

## Run

build, then

```
# bang is used here for my zsh setup
time ( build/apps/program >! output/chNa.ppm )
```
